# K12 Signature Building

A configurable email signature builder for K-12 school districts.

Created by James Thom.

## Files

- `K12_Signature_Building_Builder.html` - Open this file in a browser to configure your district and generate staff email signatures.
- `docs/K12_Signature_Building_User_Guide.pdf` - PDF directions for administrators and staff.
- `config/sample-district-config.json` - Example configuration backup file.
- `CHANGELOG.md` - Version history.
- `LICENSE.txt` - Permission and usage terms.

## Quick Start

1. Open `K12_Signature_Building_Builder.html` in Chrome, Edge, or another modern browser.
2. Go to Admin Mode.
3. Configure district name, colors, logos, buildings, phone numbers, notices, and options.
4. Use `Download Staff Generator` to create the staff-facing HTML file.
5. Share the staff-facing generator with employees.

## Logo Recommendation

For best PNG export and portability, upload logo image files instead of only linking to a hosted URL. Uploaded logos are embedded directly into the generator.
