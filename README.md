# K12 Signature Building

**K12 Signature Building** is a configurable email signature generator designed specifically for **K–12 school districts**.

It allows district administrators to create a district-branded email signature generator without writing code. Once configured, the application exports a standalone **Staff Generator** that employees can use to create consistent, district-approved email signatures.

---

## Features

* 🎨 District branding and custom color themes
* 🖼️ Upload or link district logos
* ♿ ADA/WCAG accessibility support

  * Logo Alt Text
  * HTML email signatures
  * Optional accessibility notices
* 🏫 Configurable buildings and departments
* ☎ Automatic building phone numbers
* 📠 Optional fax numbers
* 🔗 Custom website, scheduling, and meeting links
* 📱 QR Code support
* 📧 Google Workspace (Gmail) and Microsoft Outlook compatible
* 📄 Optional Public Records and Confidentiality notices
* 🔒 Administrator controls for required notices
* 👤 Multiple signature layouts
* 📋 One-click HTML signature copy
* 🖼️ PNG export
* 💾 Import and export district configurations
* 📦 Export standalone Staff Generator

## Typical Workflow

1. Configure your district branding.
2. Upload district logos.
3. Add buildings and phone numbers.
4. Customize legal notices and generator settings.
5. Export the Staff Generator.
6. Distribute the Staff Generator to district employees.

Employees simply enter their information and copy their completed signature into Gmail or Outlook.
