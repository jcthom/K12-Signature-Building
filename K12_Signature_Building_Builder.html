Email Setup Directions

Gmail / Google Workspace
  1. Click Copy Signature.
  2. Open Gmail and select Settings → See all settings.
  3. Scroll to Signature, create or select a signature.
  4. Paste using Ctrl+V or Cmd+V.
  5. Scroll down and click Save Changes.
Outlook Web / Microsoft 365
  1. Click Copy Signature.
  2. Open Outlook on the web.
  3. Select Settings → Account → Signatures.
  4. Create or edit a signature and paste with Ctrl+V.
  5. Click Save.
